# caprocharts
Visualizations / UNFCCC data about N₂O emissions from Caprolactam production

See also:

* [Some of the cleanest Polyamide and Nylon precursor Factories are shutting down (Feb
  12 2026)](
  https://industrydecarbonization.com/news/some-of-the-cleanest-polyamide-and-nylon-precursor-factories-are-shutting-down.html)
* [Visualizations of N₂O emissions from Caprolactam by Country](
  https://industrydecarbonization.com/docs/n2o/#caprocharts)

# data source

The data is extracted from the [Common Reporting Tables provided by the UNFCCC](
https://industrydecarbonization.com/docs/unfccc/).

Copyright information [according to the UNFCCC](
https://unfccc.int/this-site/terms-of-use): *"All official texts, data and documents,
including low resolution webcast transmissions, are in the public domain and may be
freely downloaded, copied and printed provided no change to the content is introduced,
and the source is duly acknowledged."*

The script itself is Open Source under a 0BSD license. It uses the Open Source [plotly](
https://plotly.com/) library (MIT license) to create the visualizations.
